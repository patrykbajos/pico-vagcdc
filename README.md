# Requirements
* Raspberry Pi Pico (W)
* Micropython
* Voltage Level Shifter

# Circuit
* DATA OUT -> GP0
* DATA IN -> SPI0_TX (GP19)
* CLOCK -> SPI0_SCK (GP18)
